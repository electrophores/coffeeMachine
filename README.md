java Создать абстрактный класс Кофемашина.
Кофемашина должна уметь наливать 4 вида кофе.
Для того чтобы налить кофе надо воспользоваться абстрактным методовм getCoffee(String coffeType);
Далее создать реализацию этого абстрактного класса.
Так же потребуется создать абстрактный класс Кофе и 4 разных вида кофе, т.е его реализаций.
Каждый кофе должен содержать объем и цену.
После в реализованной кофе машине, в зависимости от переданной строки,
 должен создаваться определённый вид кофе(т.е объект этого класса)
 и возвращаться из метода.
 Использовать реализованную кофе машину чтобы в четыре переменные Coffee вставить
 значения 4-х разных видов кофе.
